# Profile Picture Prompts for GLANCE Leurres

**Purpose:** Ideogram prompts for Nicolas to generate realistic professional headshots to replace colored-circle avatars in GLANCE leurres.

**Specs:**
- Resolution: 256x256px (displayed at 36px, high-res for Retina)
- Format: JPG
- All prompts include "1:1 square crop"
- Style: realistic professional headshot, LinkedIn-quality

## Usage Instructions

1. Open [ideogram.ai](https://ideogram.ai)
2. Paste each prompt verbatim
3. Select "Realistic" style if available
4. Generate, pick best result
5. Resize to 256x256px (Ideogram may output larger)
6. Save with the specified filename into `ga_library/leurres/`
7. Update `leurres.json` to add a `"pfp"` field pointing to the filename
