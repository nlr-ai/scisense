This resonates — graphical abstracts are often the last thing created before submission, but they're the first thing readers see when scrolling.

We built [GLANCE](https://glance.scisense.fr) specifically to address this gap. It's a scoring framework that measures whether a GA communicates its key message in 5 seconds of scrolling (the typical attention window for social media and journal TOCs).

{context_specific_paragraph}

The core insight: most GAs fail not because researchers lack design skills, but because **the visual encoding doesn't match how the human visual system processes information pre-attentively**. Position and length are processed before color and area — so a bar chart often outperforms an infographic for the same data.

GLANCE classifies GAs into diagnostic archetypes (e.g., "Encyclopedia" = too much text, "Eye Candy" = visually striking but the message doesn't transfer) with specific, actionable fixes.
