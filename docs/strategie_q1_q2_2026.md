## Décisions Checkpoints

### Gate Phase 1 → Phase 2 (Semaine 4)

**Go si:**
- [ ] ≥40% connexions LinkedIn acceptées
- [ ] ≥2 calls bookés
- [ ] ≥1 proposition envoyée
- [ ] Process documenté et reproductible

**No-Go si:**
- [ ] <20% connexions acceptées
- [ ] 0 calls bookés après 4 semaines
- [ ] Messages clairement inefficaces

**Action No-Go:** Itérer messages 2 semaines avant d'étendre

### Gate Phase 2 → Phase 3 (Semaine 8)

**Go si:**
- [ ] ≥2 clients signés (France ou BE/CH)
- [ ] Process Lemlist optimisé
- [ ] Capacité de produire contenu EN

**No-Go si:**
- [ ] <1 client signé
- [ ] Taux conversion <5%

**Action No-Go:** Focus France/BE/CH, reporter UK/DACH
