# SciSense - Automatisation Prospection

> **Objectif:** Gagner du temps sur les tâches répétitives, garder le contrôle sur les envois

---

## Le Principe

```
┌────────────────────────────────────────────────────────────┐
│                                                            │
│   AVANT                        APRÈS                       │
│   ──────                       ─────                       │
│   Écrire chaque email    →    Drafts prêts, tu envoies    │
│   Se souvenir des relances →  Rappels automatiques        │
│   Copier-coller messages  →   Templates pré-remplis       │
│   Tracker dans la tête    →   Dashboard temps réel        │
│                                                            │
│   Temps gagné: ~5-8h/semaine                              │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

---

## Les 4 Automatisations Clés

### 1. Drafts Gmail Automatiques
| Sans | Avec |
|------|------|
| Ouvrir Gmail, écrire l'email, personnaliser | Ouvrir Gmail, relire, cliquer "Envoyer" |

**Effet:** 2-3 min/email → 30 sec/email

---

### 2. Rappels Calendar Automatiques
| Sans | Avec |
|------|------|
| Se souvenir de relancer J+3, J+7, J+14 | Notification "Relancer Judit (Valneva)" |

**Effet:** Zéro prospect oublié

---

### 3. Messages LinkedIn Pré-écrits
| Sans | Avec |
|------|------|
| Réfléchir au message à chaque fois | Copier depuis le Sheet, coller, envoyer |

**Effet:** Message cohérent, gain de temps

---

### 4. Suivi Centralisé (Notion/Sheets)
| Sans | Avec |
|------|------|
| "J'en suis où avec ce prospect?" | Vue claire: statut, dernière action, prochaine étape |

**Effet:** Vision 360° du pipeline
