## Framework Global

### Philosophie: Warm Outreach via Content

```
┌─────────────────────────────────────────────────────────────────┐
│                    STRATÉGIE EN 3 PHASES                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  PHASE 1: WARM-UP (Semaine -2 à 0)                             │
│  ├── Publier contenu sur pain point du prospect                │
│  ├── Mentionner/taguer le prospect dans le post                │
│  ├── Liker et commenter ses posts                              │
│  └── Connexion LinkedIn avec note référençant le contenu       │
│                                                                 │
│  PHASE 2: ENGAGE (Semaine 1-2)                                 │
│  ├── Messages LinkedIn personnalisés                           │
│  ├── Email avec PDF de valeur                                  │
│  ├── Partage de cas client pertinent                           │
│  └── Interaction continue sur ses posts                        │
│                                                                 │
│  PHASE 3: CONVERT (Semaine 3)                                  │
│  ├── Appel téléphonique (Tier 1-2)                            │
│  ├── Email avec offre concrète                                 │
│  ├── Proposition de RDV café/call                              │
│  └── Relance finale avec nouvel angle                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Séquence Type 21 Jours

| Jour | Canal | Action | Objectif |
|------|-------|--------|----------|
| **-7** | LinkedIn | Publier post sur pain point, mentionner prospect | Visibilité, crédibilité |
| **-5** | LinkedIn | Liker/commenter 2-3 posts du prospect | Warm-up, reconnaissance |
| **-3** | LinkedIn | Republier avec commentaire, 2e mention | Renforcer présence |
| **0** | LinkedIn | Demande de connexion + note personnalisée | Premier contact direct |
| **+2** | LinkedIn | Message de suivi si accepté | Engager conversation |
| **+5** | Email | Introduction + PDF teaser | Valeur ajoutée |
| **+7** | LinkedIn | Commenter son dernier post | Maintenir présence |
| **+10** | Email | Cas client / étude pertinente | Preuve sociale |
| **+12** | LinkedIn | Partager contenu + message | Multi-touch |
| **+14** | Téléphone | Appel (Tier 1-2 uniquement) | Contact humain |
| **+17** | Email | Offre concrète + calendly | Call-to-action |
| **+21** | LinkedIn | Message final, nouvel angle | Dernière chance |
